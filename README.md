# Swift Structures Practice: Movie Library

A practical example demonstrating value types, mutating/immutable approaches, and clean Swift code.

## Concepts Demonstrated

- **Value Types vs Reference Types** (using `struct`)
- **Mutating vs Immutable Methods**
- **Clean Code Principles** (DRY, Single Responsibility)
- **Functional Programming** (`map`, `filter`, `sorted`)
- **String Formatting and Interpolation**

## Key Features

1. `Movie` struct with properties
2. `mutating func markAsWatched()` - modifies instance
3. `func withYear(_:) -> Movie` - returns new instance (immutable)
4. Unified formatting function
5. Sorting and filtering examples

## Learning Outcomes

- Understanding when to use mutating vs immutable approaches
- How value types work in Swift
- Writing clean, maintainable Swift code
- Practical use of higher-order functions

## Usage

Run in any Swift environment (Xcode Playground, Swift REPL, online compiler).
